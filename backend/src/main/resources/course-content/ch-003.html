一、电池系统层级结构

储能电池系统采用分级架构，自底向上为：单体 → 模组 (Module) → 簇 (Rack) → 集装箱/舱体。每一层级均需配置温度采样点与电气连接保护。

1.1 电池簇 (Battery Rack)

  • 由若干模组串联组成，典型电压 600V～1500V 直流
  • 配置簇级断路器、熔断器、接触器
  • 独立热管理回路（液冷板或风道）

二、BMS — 电池管理系统

BMS 是保障电池安全运行的"神经中枢"，主要功能包括：

  1. 数据采集：单体/模组电压、电流、温度、绝缘电阻
  2. 状态估算：SOC（荷电状态）、SOH（健康状态）
  3. 均衡管理：主动/被动均衡，减小单体压差
  4. 故障诊断：过压、欠压、过流、过温、温差过大等
  5. 保护执行：分级告警，必要时切断充放电回路

2.1 BMS 告警分级（典型）

级别动作示例
一级（预警）记录、通知运维单体温度 > 45℃
二级（告警）降功率运行单体温度 > 55℃ 或压差过大
三级（保护）切断充放电单体温度 > 60℃ 或绝缘故障

三、PCS — 储能变流器

PCS 负责电池直流侧与电网交流侧的能量转换。安全相关功能包括：防孤岛保护、直流过压/过流保护、与 BMS 的联动停机。运维时应定期检查 PCS 告警记录与散热系统。

四、EMS — 能量管理系统

EMS 执行站级调度策略（削峰填谷、AGC、AVC），汇总各簇 BMS 数据，生成运行报表。安全方面需确保 EMS 与消防、门禁系统的联动逻辑正确配置。

五、辅助系统

  • 热管理：维持电池工作温度 15～35℃，避免局部过热
  • 消防：烟感/温感/可燃气体探测 + 气体/细水雾灭火
  • 通风：事故后排烟、正常运行散热
  • 监控：视频监控、环境温湿度、水浸检测
各子系统并非孤立运行。典型安全链：BMS 检测异常 → PCS 停机 → EMS 记录并推送告警 → 消防系统待命/启动。

六、本章要点

理解电池系统层级；掌握 BMS/PCS/EMS 核心功能与告警分级；了解辅助系统在安全保障中的角色。

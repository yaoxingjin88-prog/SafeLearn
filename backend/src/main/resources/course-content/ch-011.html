一、消防联动逻辑

储能电站消防联动是指：当火灾报警系统确认火情后，自动触发灭火、排烟、门禁、广播等子系统协同动作。

1.1 典型联动序列

  1. 烟感 + 温感 / 气体探测复合确认（防误喷）
  2. 声光报警，广播疏散指令
  3. 切断故障区域直流电源（BMS/PCS 联动）
  4. 关闭通风防火阀，启动排烟风机
  5. 延时 30s 后释放灭火剂（人员撤离时间）
  6. 灭火后持续监测温度，防止复燃

二、应急演练要求

  • 频次：每半年至少一次消防联合演练
  • 内容：报警确认、疏散路线、灭火系统手动启动、与消防队交接
  • 记录：演练方案、签到表、总结报告、整改闭环

三、演练常见问题

问题改进
人员不熟悉手动启动按钮位置张贴操作流程图，每季度实操培训

联动逻辑仅测试报警，未测灭火年度全要素实战演练

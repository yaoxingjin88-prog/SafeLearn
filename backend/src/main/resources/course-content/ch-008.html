一、BMS 系统架构

电池管理系统 (Battery Management System) 采用分布式采集 + 集中式控制架构：

  • BMU / CSC：从控/采集板，贴近模组采集单体电压与温度
  • BCU / BAMS：主控，汇总数据、执行保护策略、对外通信
  • 通信网络：CAN / RS485 / Daisy Chain 菊花链

二、核心功能模块

2.1 数据采集

  • 单体电压（精度 ±5mV）、模组温度（精度 ±1℃）
  • 簇电流（霍尔传感器）、绝缘电阻（≥500Ω/V）
  • 采样周期通常 100ms～1s

2.2 SOC / SOH 估算

SOC 常用安时积分法 + 开路电压校正；SOH 通过容量衰减或内阻增长评估。SOH 低于 80% 的电池应列入更换计划。

2.3 均衡管理

  • 被动均衡：高压单体通过电阻放电，成本低
  • 主动均衡：能量在单体间转移，效率高，大型储能更常用

2.4 热管理联动

BMS 根据温度分布控制液冷泵速、风机转速，维持模组温差 ≤ 5℃。

三、BMS 与站控系统接口

接口数据/指令
BMS → PCS充放电允许、功率限制、紧急停机
BMS → EMS全站电池状态、告警信息、历史曲线
BMS → 消防热失控预警信号（可选硬接线）

了解 BMS 分层架构与四大功能模块；理解 BMS 在储能安全链中的枢纽地位。

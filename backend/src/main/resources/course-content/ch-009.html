一、告警阈值设置原则

BMS 告警阈值应结合电池厂家规格书、当地气候及运行策略制定，并每年复核。原则：宁可误报，不可漏报。

1.1 温度告警参考值（磷酸铁锂）

级别单体温度动作
预警≥ 45℃记录、通知值班员
告警≥ 55℃降功率 50%，加强监控
保护≥ 60℃切断充放电，启动应急预案

1.2 电压与压差

  • 单体过压：≥ 3.65V（充电）→ 停止充电
  • 单体欠压：≤ 2.50V（放电）→ 停止放电
  • 簇内压差：≥ 50mV 持续 10min → 告警，检查均衡

二、告警响应流程 (SOP)

  1. 接收：EMS/短信/声光告警，确认告警级别与位置
  2. 研判：查看温度/电压曲线趋势，排除传感器故障
  3. 处置：按级别执行降功率、停机、现场检查
  4. 记录：填写告警处置单，附截图与操作时间
  5. 复盘：一级以上告警 24h 内提交分析报告

三、常见误报与排查

  • 温度传感器接触不良 → 读数跳变，检查接线
  • 通信中断导致数据冻结 → 检查 CAN 终端电阻
  • 环境温度骤变 → 对比舱内多点位温度
深圳事故教训：55℃ 告警后若仅"加强观察"而不降功率，3 分钟内可升至 75℃ 进入不可逆阶段。

四、本章要点

掌握温度/电压告警阈值；能执行标准响应 SOP；会区分真实告警与误报。

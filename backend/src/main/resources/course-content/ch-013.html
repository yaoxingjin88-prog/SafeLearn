一、异常分级定义

等级定义响应时限
一般异常不影响安全运行，如通信闪断、轻微压差24h 内处理
重要异常影响性能或存在潜在风险，如冷却故障4h 内处理
紧急异常直接威胁安全，如温度超限、冒烟立即响应

二、标准处置流程

  1. 发现与报告：巡检或告警发现异常，立即报告值班长
  2. 初步研判：调取 BMS 曲线、视频监控，确认异常真实性
  3. 隔离措施：必要时停机、断开故障簇、设置警戒区
  4. 专业处置：按预案执行，厂家/消防支援
  5. 恢复与验证：故障排除后，逐步恢复并网，加强监控 72h
  6. 归档复盘：填写异常报告，更新风险库

三、典型场景处置要点

3.1 单体温度偏高

降功率 → 加强冷却 → 若 30min 未回落则停机检查 → 考虑更换异常单体

3.2 绝缘电阻下降

立即停机 → 排查直流侧绝缘破损、凝露 → 烘干除湿后复测

3.3 电池冒烟

立即切断电源 → 启动消防 → 疏散人员 → 禁止进入 → 报 119

四、信息报送

  • 紧急异常：15min 内电话报告站长及公司安全部门
  • 2h 内提交书面快报
  • 事故等级按《生产安全事故报告和调查处理条例》判定
